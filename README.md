# HTK Pet Clinic
Một ứng dụng web để quản lý hoạt động của phòng khám thú y, bao gồm lưu trữ hồ sơ thú cưng, lên lịch hẹn và theo dõi hỗ sơ, lịch khám,...

## Technologies Used
- **Backend**: Spring Boot, Java
- **Frontend**: (HTML/CSS)
- **Database**: MySQL
- **Build Tool**: Maven
- **IDE**: Eclipse
